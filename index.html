JSafe

Keep track of your passwords in a secure way

Details:

  • Written in Python3
  • Tkinter, Python GUI package
  • Secures passwords using AES encryption and SHA256 hashing algorithm
  • Stores encrypted passwords using SQLite database
  • Connects to DB using SQLAlchemy ORM

How does it work?

  • User must log in using a master password
  • The hash of the master password, along with a random initializaton vector, is used to encrypt all passwords.
  • Passwords stored in database in binary form
  • Even if database is compromised, ciphers cannot be decrypted without master password

Imapp

Instant messaging desktop app

Details:

  • Written in Python3
  • Tkinter, Python GUI package
  • Multi-threading to send messages, receive messages and run GUI simultaneously
  • Connections created using Socket module

How does it work?

  • Server listens for incoming connections
  • Once client connects using GUI, connection created and client is registered.
  • Client incoming messages served on new thread, which are then broadcasted to all connected clients
  • Closes connection and thread once a client disconnects
